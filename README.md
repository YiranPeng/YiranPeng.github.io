# Yiran Peng Personal Website

Static personal website for applications, job search, CV, research projects, publications, and portfolio.

## Preview

Open `index.html` directly in a browser.

## Current Assets

- `assets/photo.jpg`: personal portrait
- `assets/portrait-playful.png`: playful transparent portrait used on the home page
- `assets/portrait-hero.png`: current hero portrait
- `assets/hero-doodle.svg`: pink-blue doodle hero background
- `resume.pdf`: downloadable CV

## Current Features

- English / Chinese language switch in the top-right navigation
- Project overview cards with detail popups, including AI BJD Diary and Triton / FlashAttention GLM-ASR optimization
- Publication cards link to IEEE pages
- Personal interests entry page: `interests.html`
- Real country-boundary travel map with visited countries highlighted, defaulting to Europe view with drag and zoom controls
- Dedicated travel log page: `travel.html`, with calendar-style country cards and placeholder images/text

Visited countries currently highlighted on the travel map: China, United Kingdom, Spain, Iceland, Belgium, Czech Republic, Hungary, Austria, France, Norway, Poland, Denmark, and Netherlands.
